Kleurmodellen

RGB-model

Het RGB-model gebruikt licht om kleuren te maken. RGB staat voor Rood, Groen en Blauw. Door deze drie kleuren in verschillende hoeveelheden te mengen, kan de computer bijna alle kleuren weergeven die je op een scherm ziet.

Voorbeelden:

  • Wit: Rood 255, Groen 255, Blauw 255
  • Zwart: Rood 0, Groen 0, Blauw 0
  • Rood: Rood 255, Groen 0, Blauw 0
  • Groen: Rood 0, Groen 255, Blauw 0
  • Blauw: Rood 0, Groen 0, Blauw 255

    • CMYK-model

    Het CMYK-model gebruikt inkt om kleuren te maken. CMYK staat voor Cyaan, Magenta, Yellow en Key (zwart). Dit model wordt gebruikt bij printers. Door verschillende hoeveelheden van deze vier kleuren te mengen, kan de printer veel verschillende kleuren afdrukken.

    Voorbeelden:

  • Wit: Cyaan 0, Magenta 0, Yellow 0, Zwart 0
  • Zwart: Cyaan 0, Magenta 0, Yellow 0, Zwart 1
  • Rood: Cyaan 0, Magenta 1, Yellow 1, Zwart 0
  • Groen: Cyaan 1, Magenta 0, Yellow 1, Zwart 0
  • Blauw: Cyaan 1, Magenta 1, Yellow 0, Zwart 0

    • Belangrijk om te weten

    RGB wordt vooral gebruikt voor schermen zoals computers, telefoons en tv's. CMYK wordt vooral gebruikt voor printen. Het verschil zit dus in het materiaal: licht versus inkt.
